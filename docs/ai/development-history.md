# Development History

## 開発状態

2026-06-23 時点では開発継続中。v1.11.0 の開発凍結は v1.12.0 で解除し、以降は通常どおり機能追加・変更・バグ修正を行う。開発作業は `AGENTS.md` のルールに従い `feature-dev` 経由で実施する。

改めて凍結する場合は、この履歴と `AGENTS.md` の入口ルールを更新する。

## ライブ編集ポリシー（v1.12.0 以降）

- 標準 Markdown 記法（CommonMark + GFM）はライブ編集対象。カーソル行で生記法、他行で装飾表示する。
- HTML タグ（`<details>` アコーディオン等）はビューア専用。常にウィジェット描画し、編集は標準ソースエディタで行う。

## v1.15.0 から v1.15.2 の主な内容

- ブロックウィジェット直下のクリック位置ずれを根本修正した。`toDOM` の `requestMeasure` 撤去、`estimatedHeight` 動的化、テーブルセル `line-height` 固定を含む（R-28-10 / R-28-11）。
- 上下矢印がテーブルやアコーディオンをまたぐ際の複数行ジャンプを修正した。`ArrowUp` / `ArrowDown` カスタムコマンドを使う（R-28-12）。

## v1.12.0 から v1.14.x の主な内容

- 表と `<details>` アコーディオンをビューア専用化した（R-22-02 / R-27-03 再定義、R-28-09）。
- アコーディオンの開閉状態を summary キーで保持した（R-27-02）。
- チェックボックス ON/OFF のホストから Webview への反映を修正した（R-08-07）。
- 相対パスリンクの山括弧宛先 `[ラベル](<...>)` を `openLink` で除去した（R-21-03）。
- 本文余白を `20px 40px 24px 48px` へ調整した（R-28-07）。

## v1.7.0 から v1.11.0 の主な内容

- 表セルのインライン記法（太字・斜体・インラインコード）を描画した（R-22-03、`appendInlineCell`）。
- ビュー切り替えを同一タブ内の再描画方式へ変更し、保存確認ダイアログを出さないようにした（R-03 再定義）。
- 折りたたみ対象を見出しから HTML `<details><summary>` アコーディオンへ再定義した（デフォルト折りたたみ、R-27 再定義）。
